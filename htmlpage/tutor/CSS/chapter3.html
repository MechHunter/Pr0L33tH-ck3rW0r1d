The code in css is the core component, it is fairly simple as you dont need to type much to get an outcome and also it is quite quick to learn.

breaking down the image

in this image, after we see the tags, we see some lines of code, lets go through them!

background-color:

background-color changes the background's color of whatever the tag is, for instance if i change the background color of "body" then the body section's background color changes.

color:

color changes the color of the text in whatever the tag is, so if the tag is h1 and the code is color: red; then the color of all heading 1s gets changed to red.

padding:

padding changes the space between the text and the boarder of whatever the tag is, so if the tag is set to body and you type padding: 10px; then the text will get pushed closer to the center of the page by 10 pixels.

margin:

margin changes the space outside of the boarder, so if you typed margin: 10px then nothing would be able to get closer then 10 pixels.

border:

border adds an boarder to whatever the tag is, so if the tag is img and you type border: 1px solid black; then your image will be surrounded by a solid black boarder, 1 pixel thick.

text-decoration:

text-decoration adds text decorations like bold text, underlines, ect. So if the tag was h1 and you typed in text-decoration: underline; then all heading 1s get underlined.

one last thing to note, when ending a line of code make sure you end it with a ; otherwise the web browser doesn't know when the code as finished.
